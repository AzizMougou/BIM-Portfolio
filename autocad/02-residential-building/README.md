# Project 02 — Multi-Room Apartment

## Overview
A complete 2D floor plan of a multi-room apartment drawn in AutoCAD.
This is the second project in my BIM portfolio, building directly 
on the fundamentals learned in Project 01. The key focus of this 
project was planning all dimensions before drawing, resulting in 
a cleaner and more precise technical drawing.
