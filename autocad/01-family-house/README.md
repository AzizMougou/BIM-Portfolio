# Project 01 — Family House Floor Plan

## Overview
A complete 2D floor plan and section view of a family house, 
drawn in AutoCAD as the first project in my BIM portfolio journey.
This project focused on mastering the fundamental tools of AutoCAD
and producing a professional architectural drawing set.

## Drawings Produced
- Ground Floor Plan at 1:100
- Section A-A at 1:40

## Rooms
- Master Bedroom
- Children's Room
- WC
- Bathroom
- Entrance
- Living Room
- Dining Area
- Kitchen

## Tools & Commands Learned
- LINE, OFFSET, TRIM — basic geometry
- LAYER — layer management and organization
- OSNAP — precise snapping to endpoints and intersections
- HATCH — wall fill patterns
- DIMLINEAR, DIMCONTINUE, QDIM — dimensioning
- TEXT, MTEXT — room labels and annotations
- MVIEW — viewport creation in layout
- PLOT — exporting to PDF

## What I Learned
This was my first AutoCAD project. I learned how to organize 
a drawing using layers, draw walls with correct thickness, 
place doors and windows, add furniture, and set up a 
professional layout with title block and correct scale.
I also learned the importance of OSNAP for drawing accuracy
and how to use paper space for printing.

## Tools Used
- AutoCAD 2024
- DWG To PDF for export
